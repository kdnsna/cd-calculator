# 3年期大额存单转让计算器

## 在线访问地址

### 国内访问推荐
- **GitHub Pages**: https://kdnsna.github.io/cd-calculator/
- **Vercel**: https://cd-calculator-theta.vercel.app

### 备用地址
- 自定义域名（配置中）: cd.kdnsna.cn

## 使用方法

1. 在微信中打开上述链接
2. 输入6个必填项：
   - 转让挂牌利率
   - 存单面值
   - 存单利率
   - 存单现持有天数
   - 购买价格
   - 本人计划持有天数
3. 点击"立即计算"
4. 可复制结果分享给朋友

## 功能特点

- 📱 完美适配手机端

- 🎯 输入验证 + 错误提示
- 📋 一键复制结果
